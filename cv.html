Habilidades

Possuo noções de PhotoShop, o utilizando para extrair os assets do layout que recebo. Também como parte do meu workflow utilizo a técnica Kanban e o WordPress como gerenciador de conteúdo.

O framework CSS que escolhi trabalhar é o Bootstrap, o utilizo de forma semântica com mixins ao invés de poluir o HTML. Para me ajudar nessa tarefa utilizo o Sass como pré-processador e o Gulp para automatizar algumas tasks.

Tenho uma noção de JQuery e atualmente estou estudando JavaScript puro, pois acredito que antes de aprender qualquer framework ou biblioteca há a necessidade de conhecer a linguagem em si e entendê-la. Outra tecnologia que gosto muito e tenho dedicado um tempo de estudo é o SVG, além de ser mais leve que os outros formatos de imagens ele nos possibilita efeitos incríveis.
